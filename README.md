# TeamSync

TeamSync es una aplicación web diseñada para mejorar la coordinación de trabajos grupales universitarios. Centraliza en un solo lugar la gestión de tareas, recursos y progreso del equipo, evitando la pérdida de información en chats y facilitando el seguimiento del avance de cada integrante mediante un dashboard, lista de tareas y barra de progreso.
